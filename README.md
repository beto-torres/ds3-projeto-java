## 📋 Pré-requisitos

Antes de começar, certifique-se de ter as seguintes ferramentas instaladas na sua máquina:

1.  [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) - Versão 25 ou superior.
2.  [Git](https://git-scm.com/) instalado no seu computador.
3.  [VS Code](https://code.visualstudio.com/) com a extensão **Extension Pack for Java** instalada.
4.  [Docker](https://www.docker.com/) (e Docker Desktop, se estiver no Windows/Mac) rodando na máquina para o banco de dados.
